 <h2 id="1_3"><a href="#1_3">#</a>IP查询</h2>
  <p>本文整理于网络<p>
  <p>关于IP查询的API接口，很多的第三方平台提供的有免费的API接口，通过网上的搜素找到了一些，这里为大家整理一下。<p>
   <a href="http://www.binstd.com/api/ip.html">进制数据</a>
  <p>BinSTD进制数据的IP查询API接口是免费提供给大家使用的，不存在支持JSON、JSONPl两种格式，同时也支持php、python、java三种语言开发。<a href="http://www.binstd.com/api/ip.html">http://www.binstd.com/api/ip.html</a></p>
  <a href="https://github.com/wzhe06/ipdatabase">ipdatabase</a>
  <p>这是我在github上找到的找到的用二叉树快速查找IP地址数据库。其原理是利用二叉树实现IP查询，首先将10进制IPV4地址转化为二进制构建二叉树，利用二叉树搜索进行搜索，查询时间复杂度log2n，比传统IP库Ñ的查询速度高出一个量级。<a href="https://github.com/wzhe06/ipdatabase">https://github.com/wzhe06/ipdatabase</a></p>
  <a href="http://ip-api.com/">IP-API</a>
  <p>这是一个国外的API查绚接口，免费用于非商业用途，无需API密钥，易于集成，提供JSON，XML，CSV，Newline，PHP。IP-API将一些经常问道的问题放在页面首页，帮助大家解决疑惑。<a href="http://ip-api.com/">http://ip-api.com/</a></p>
  <a href="http://pv.sohu.com/cityjson">搜狐IP查询</a>
  <p>这个搜狐IP查询是我在CSDN上看到的，觉得比较不错就在这里也推荐一下。</p>
  <p>搜狐IP地址查询接口（默认GBK）：<a href="http://pv.sohu.com/cityjson">http://pv.sohu.com/cityjson</a></p>
  <p>搜狐IP地址查询接口（可设置编码）：<a href="http://pv.sohu.com/cityjson?ie=utf-8">http://pv.sohu.com/cityjson?ie=utf-8</a></p>
  <p>搜狐另外的IP地址查询接口：<a href="http://txt.go.sohu.com/ip/soip">http://txt.go.sohu.com/ip/soip</a></p>
  <a href="https://www.juhe.cn/docs/api/id/1">聚合数据</a>
  <p>聚合数据同样支持免费的IP地址查询，支持JSON格式，支持JavaScript、C#、GO、C、PHP、Python语言开发。<a href="https://www.juhe.cn/docs/api/id/1">https://www.juhe.cn/docs/api/id/1</a></p>
  <a href="https://www.avatardata.cn/Docs?keyword=IP">阿凡达数据</a>
  <p>可以查询外网IP信息和IP地址，支持JSON和XML格式。<a href="https://www.avatardata.cn/Docs?keyword=IP">https://www.avatardata.cn/Docs?keyword=IP</a></p>

  <h2 id="1_4"><a href="#1_4">#</a>全国天气预报</h2>
  <p>本文整理于网络<p>
  <p>为方便开发者学习和实践，从网上整理了比较全面的免费天气API接口。整理的天气接口均来自产品平台开放接口。<p>
  <a href="http://www.binstd.com/api/">进制数据</a>
  <p>BinSTD进制数据提供的有天气预报查询接口，全国3000多个省市的天气预报查询，包括实时天气气温、最高最低温度、风级、风力、湿度、气压，穿衣、运动、洗车、感冒、空气污染扩散、紫外线等指数，7天天气、风力、最低最高温度、日出日落时间，未来24小时的时天气、气温，空气质量指数、PM2.5指数等信息。不管是对于企业还是个人来说5元10000次使用还是特别低价的，我在这里也把这个接口推荐给大家。<a href="http://www.binstd.com/api/">http://www.binstd.com/api/</a></p>
  <a href="http://data.cma.cn/Market/MarketList.html">中国气象数据网</a>
  <p>中国气象数据网服务提供的气象API服务主要包括11大类。气象API服务以Restful WebServices的方式提供，对于要素型数据以Json格式直接返回，对于文件型数据以Json格式返回文件清单列表，通过文件清单的文件URL进行文件下载。普通用户需要认证成个人实名注册用户或教育科研实名用户。每类资料的API服务仅提供一次免费7天使用，供接口调用调试与验证。对个人开发者或者学生来说，只有七天的试用期，还是要收费的。具体使用情况，还是需要大家自行来选择。<a href="http://data.cma.cn/Market/MarketList.html">http://data.cma.cn/Market/MarketList.html</a></p>
  <a href="https://dev.heweather.com/docs/api/weather-gird">和风天气</a>
  <p>和风天气对个人开发者、学生、公益组织以及所有非商业环境使用而推出的和风天气开发者计划，即和小风免费天气数据服务，通过申请后即可免费使用。相比较于商业版本，免费版本在功能上略少，但对于个人开发者学习使用还是比较方便的。<a href="https://dev.heweather.com/docs/api/weather-gird">https://dev.heweather.com/docs/api/weather-gird</a></p>
  <a href="https://www.seniverse.com">心知天气</a>
  <p>心知天气也提供的有免费的API接口，主要涉及国内主要370个城市，包括天气实况，包括天气现象文字、代码和气温 3 项数据；未来 3 天天气预报，包括白天天气现象文字及代码、晚间天气现象文字及代码、当天最高温度和最低温度、风向风速；6 项基本类生活指数，包括穿衣、紫外线强度、洗车、旅游、感冒、运动指数。<a href="https://www.seniverse.com">https://www.seniverse.com</a></p>
  <a href="https://github.com/overtrue/weather">高德开发平台</a>
  <p>高德地图不仅开发了地图的接口，也开放了天气接口，在使用API之前需要注册账号申请key。
除此之外，我还在github上找到了基于高德开放平台接口的 PHP 天气信息组件。当然在使用此接口之前也需要在高德平台注册获取密钥。<a href="https://github.com/overtrue/weather">https://github.com/overtrue/weather</a></p>
